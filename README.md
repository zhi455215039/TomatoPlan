# TomatoPlan
计划效率app



这是一款简约的计划记录工具，可以记录下过去以及未来每天的计划表，方便观察自己进步的点点滴滴。同时还能记录亲友的生日以及自己的各种纪念日哦，自带提醒功能，再也不用担心忘记啦！当然番茄计划还自带番茄时钟哦，番茄工作法让你效率翻倍~~


 这是主界面,是日历,上面有显示农历公历以及各种节日时节，同时在添加当日计划后上面会显示前四条计划
